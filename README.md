# Cognitive Load Analyzer for Students

This project is a web-based system that analyzes cognitive load using quiz performance.

## Features
- User login system
- Quiz-based evaluation
- Cognitive load analysis (Low, Medium, High)
- Result generation and report storage

## Technologies Used
- Python
- HTML, CSS, JavaScript
- SQLite

## How It Works
The system calculates cognitive load using a rule-based approach based on:
- Accuracy
- Time taken

## Future Scope
- Subject-wise quizzes
- PDF-based quiz generation
